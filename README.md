# 🎓 Graduator

**Student Intelligence System** — A beautifully designed, all-in-one academic calculator for students.

🌐 **Live Site:** `https://<your-username>.github.io/graduator`

---

## ✨ Features

- **⚡ Attendance Tracker** — Interactive sliders, ring gauges, spectrum bar, bunk allowance calculator
- **📋 All Subjects** — Per-subject attendance tracking with safe/warning/danger analysis
- **🎓 GPA / CGPA** — Supports both 10-point (Indian) and 4-point (US) grading scales, CGPA tracker, target planner
- **🗓 Study Planner** — Exam countdown, subject load distribution, Pomodoro session planner
- **📖 Scholar's Codex** — 12 science-backed study tips + live unit converters

## 🎨 Design

- Deep-space holographic UI with particle network background
- Shooting stars animation
- True CSS 3D perspective tilt on every card
- Custom glowing cursor with smooth ring
- Glassmorphism panels with animated shimmer bars
- Fully responsive

## 🚀 Deploy on GitHub Pages

1. Create a new repository on GitHub (e.g. `graduator`)
2. Upload `index.html` and `README.md`
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)`
5. Click **Save**
6. Your site will be live at `https://<your-username>.github.io/graduator`

## 🛠 Local Usage

Just open `index.html` in any modern browser — no build steps, no dependencies, no server needed.

```bash
# Clone and open
git clone https://github.com/<your-username>/graduator.git
cd graduator
open index.html
```
